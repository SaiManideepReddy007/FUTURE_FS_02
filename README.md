# FUTURE_FS_02
# 🏠 NestCRM - Client Lead Management System

A full-stack Mini CRM built for hostel businesses to manage student inquiries.

## 🔗 Live Links
- **Frontend:** https://future-fs-02-iota-three.vercel.app/
- **Backend API:** https://nestcrm-backend.onrender.com/

## ✨ Features
- Public inquiry form for students
- Secure admin login with JWT
- Dashboard to view and manage all leads
- Lead status tracking: New → Contacted → Converted → Lost
- Add notes for each lead
- Delete leads
- Filter leads by status

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite), Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** JWT

## ⚙️ Setup Instructions

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
